# Efe-Shippuden Re-zero Community Website

An interactive website made to show artwork, videos, and stories for the supernatural comic series "Efe-Shippuden."

## Features
Semantic HTML5 Layout:** Utilizes structured layout elements (`<nav>`, `<section>`, `<aside>`) for clean SEO and accessibility.
Multimedia Integration:** Supports HTML5 video and audio players for custom series animations and soundtracks.
Social Connectivity:** Built-in pathways to bridge the community with Discord, Wattpad, and Instagram.

## Technologies Used
* HTML5
* CSS3 (Murikami.css)
